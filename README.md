# Real-Deal-Football-Quiz
#Football Quiz Game

A graphics based Football quiz game programmed in C++ and SFML. Players must answer Football related questions throughout
an interactive graphical user interface and are score based on correct answers. This project demonstrates C++ programming with graphics,
use of loops, conditionals.
